# Flask + PostgreSQL + Nginx

Небольшое веб-приложение, развернутое с помощью Docker Compose.

## Используемые технологии

- Docker
- Docker Compose
- Flask
- PostgreSQL
- Nginx

## Запуск

docker compose up -d

## Остановка

docker compose down

## Проверка

http://localhost

## Переменные

DB_HOST
DB_NAME
DB_USER
DB_PASSWORD
DB_PORT
